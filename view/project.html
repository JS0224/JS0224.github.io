BurgerKing Auto Survey

Burger King experience survey site (kor.tellburgerking.com) allows customers to respond to survey questions about specific Burger King branches. Users can start a survey with the code given on receipts. When finished answering all the questions, a final code appears, which can be used with receipt to get free meal upgrade.

Development Environment

  • Python 3.7.3
  • Selenium 3.141.0

    • Description
